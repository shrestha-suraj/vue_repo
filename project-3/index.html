When we update any data in the vue data object, this whole componenet gets re-renderd. If we are running an expensive funtion
during this time and this function doesnot depend on the data that is being manipuated, the cost could be expensive for an
enterprise level application. Hence we want to preserve an information about a variable in data object if it is not depenende.
Example:
This value depends on a data object called num1. It changes by 1 per button click: {{getNum1}}
 
This value depends on a data object called num2. It increase by 1 per button click: {{getNum2()}}

Look at the console. The buttons trigger a function that chage num1 and num2 but if we trigger num2 it doesnot rerender num1 function
for num 1 as well. To prevent we put the function for num1 in computed object of Vue.
